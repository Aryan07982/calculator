Main Features

  1. Calculates only a pair of numbers at a time.
  2. Decimal numbers are rounded up to 3 decimal places.
  3. Pressing an operator or equal sign before entering numbers is disabled.
  4. Pressing “AC” completely resets the calculation.
  5. Divide by 0 returns “Infinity” instead of an “Error”.
  6. Pressing operator keys consecutively reassigns the operator to the new operator.
  7. After the result is displayed (pressing “=” or Enter), entering new numbers resets the calculation.

Additional Features

  1. Users can enter “.” only once for any number.
  2. Backspace button “C” is added.
  3. Keyboard support for number keys, “AC” (Escape), “C” (Backspace), “=” (Enter).
